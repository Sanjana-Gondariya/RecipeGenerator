Recipe Generator
A full-stack web application that helps users discover recipes based on ingredients they have on hand.
Features

User Authentication — Sign up, log in, and maintain a personal account
Ingredient Tracking — Add and manage ingredients you have available
Recipe Search — Browse and search recipes by name or category
Smart Recommendations — Get recipe suggestions based on your saved ingredients
Bookmarks — Save favourite recipes for quick access later

Tech Stack

Frontend: React, CSS3
Backend: Node.js, Express.js, JWT Authentication (bcrypt password hashing)
